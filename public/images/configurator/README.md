# Configurator placeholder images

Photos grouped by rifle part category. Served at `/images/configurator/{category}/` via sync to `public/images/configurator`.

| Folder | Configurator step |
|--------|-------------------|
| `actions/` | Action |
| `stocks/` | Stock / Chassis |
| `camo/` | Stock Paint |
| `triggers/` | Trigger |
| `barrels/` | Metal Finish |
| `scopes/` | Scope |
| `rings/` | Rings |
| `muzzle-brakes/` | Muzzle Brake |
| `suppressors/` | Suppressor |
| `cases/` | Case |

Add or replace files in the matching folder — update paths in `src/data/configurator-options.ts` if filenames change.
